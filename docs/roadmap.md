# SecureBank – Product Roadmap

## Current Version: v1.4.0

### ✅ v1.0.0 – Core Banking
- User registration and authentication
- JWT + httpOnly cookie auth
- Account balance management
- Basic dashboard UI

### ✅ v1.1.0 – Virtual Cards
- Virtual card generation (Visa/Mastercard)
- Encrypted card numbers and CVV
- Card management (activate/deactivate)
- Spending limits

### ✅ v1.2.0 – Transfer PIN
- Secure transfer PIN setup
- bcrypt-hashed PIN storage
- PIN verification on every transfer

### ✅ v1.3.0 – Two-Factor Authentication
- TOTP-based 2FA (Google Authenticator)
- QR code generation for setup
- Secure tempToken flow for verification

### ✅ v1.4.0 – Admin Dashboard
- User management
- Transaction monitoring
- Card overview
- Audit log viewer
